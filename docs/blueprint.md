# **App Name**: Tickety Boo

## Core Features:

- Login Page: Role-based login (Admin, User) with username/password authentication.
- Dashboard: Admin dashboard displaying all user requests with status and actions.
- Request Submission: User request form for submitting device issues with priority.
- Request Management: Admin actions to manage requests: mark as done, cancel, or archive.
- Authentication: Secure authentication to redirect to Login Page if not logged in.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to inspire confidence.
- Background color: Very light blue (#E5E9F2), almost white, providing a clean backdrop.
- Accent color: A vivid violet (#A362FF) will distinguish interactive elements without clashing.
- Font: 'Inter' (sans-serif) for a modern, machined, neutral look across the platform for headings and body text.
- Clean and modern layout for the dashboard with clear table presentation.
- Simple, recognizable icons for actions (done, cancel) and priority levels.
